PycraftPython

To improve my Python skills I developed a game for the terminal with Olls. Pycraft is a text based game inspired by Minecraft. It is quite advanced so far, with the following features:

  • Random terrain generation with ores, hills and trees
  • Auto-saving world and multiple save support
  • Chunk saving to improve performance
  • Breaking and placing blocks
  • Inventory management
  • Custom world options

We plan on taking the project further, with features like crafting and implementing other block types. Download the latest version here and try it out. It runs best on UNIX with Python 3 but will also work on Windows CMD.

PropViewPHP/JavaScript

My first PHP applications were not at all modular and did not give the user a good experience, requiring them to reload the page after every form action. These projects needing code refactoring all the time due to the increase in complexity of the site and unforeseen changes. PropView was one of the first sites I built with the mindset to keep it as modular as possible, providing easy maintenance as it grew.

PropView is a virtual property management system. This means that property owners can log the progress of their construction projects, and their clients can view photos and maps to prevent the need for them to travel to the location in person.
